SU25T Cold start & Taxi

Home

  • Radio: request startup

    \ Radio

    F5 Ground crew

    F1 Request startup

    F1 Request startup

  • APU power on

    rs L

  • Silence RWR

    in case it is already beeping:

    rs R mute or

    ra < ra > volume up/down

  • Throttles to idle

    otherwise you will flood engines and start a fire.

    rs End to stop engines

  • Start engines

    rs Home both engines

    left: ra Home, right: rc Home

  • Wait for engines idle

    will spool higher

    wait for 32% RPM

  • Radio: Request Taxi to Runway

    \ Radio

    F5 ATC

    F1 Request Taxi to Runway

  • Airport map for taxiway

    F10

    rc F8 centers on player

    mousewheel to zoom

    or use maps here

  • Navigation lights on

    rc L until blinking

    blinking: "intend to move"

    solid: "parked"

  • Taxi lights on

    ra L once for day, again for night

  • Start taxiing

    throttle up briefly, start rolling

    throttle back to 40%

    keep taxiing speed <30 km/h

    must stay on taxiways

    W wheel brake, hold if needed

  • Hold short of runway

    Check runway clear

    no inbound traffic

  • Request take off

    \ Radio

    F1

    Wait for

    "Clear for take off"

SU25T Take off

Home

  • Align on runway

  • Wheel brakes on

    hold W until stopped

  • Power up to 75%

  • Release wheel brakes

    release W

  • Full power

  • Takeoff

    @180 km/h lite, 220 km/h heavy

    15 m/s climb rate

  • Gear up

    G

  • Radio wingman "airborne"

  • Flaps up

    @330 km/h

    ls F if landing flaps, F if regular

  • Taxi lights off

    ra L

  • Navigation mode to: Enroute

    1 until "ENR" in HUD

  • If needed: select waypoint

    lc ~

  • Cruise speed 550 km/h

    or as indicated

    by flight director circle in HUD

    above speed

SU25T Return to Base

Home

  • Navigation mode to: Return

    1 until "RTN" mode in HUD

  • Select airport

    ls ~

    airport ID in HUD

  • Radio ATC "request inbound"

    \ Radio

    F5 ATC control

    Fx select airport

    request inbound

  • Follow flight director

    to return waypoint

    ignore suggested altitude (buggy)

    use 1000-1500m altitude

  • Slow to 400 km/h

  • Navigation lights on

    rc L until blinking

  • Navigation switches to: Landing

    "LNDG" in HUD

  • Prepare for final

    turn towards runway

    follow flight director

  • Slow to 280 to 320 km/h

  • Gear down

    F

Glossary

Home

  • A-A

    Air-to-Air

  • A-G

    Air-to-Ground

  • AAA

    Anti-Aircraft Artillery

  • AAP

    Auxiliary Avionics Panel. Provides power to the CDU and EGI, and controls which mode the TAD displays waypoints with. Manual

  • AAS

    Air-to-Air Submenu

  • ACP

    Armament Control Panel

  • ADF

    Automatic Direction Finder

  • ADI

    Attitude Director Indicator cockpit instrument (artificial horizon)

  • AGL

    height above ground level (cmp ASL above sea level)

  • AGM-65

    Maverick IR/EO self guided

  • AGM

    Air-to-Ground Missile

  • AH

    Attack Helicopter, for example AH-64 Apache

  • AHCP

    Armament HUD Control Panel Manual

  • AIM

    Air Intercept Missile

  • AIM-9M

    Sidewinder air-to-air missile

  • ALO

    Air Liason Officer: senior TACP member attached to a ground unit who functions as primary advisor on air operations

  • Alphabet

    NATO Phonetic alphabet:

    A Alpha

    B Bravo

    C Charlie

    D Delta

    E Echo

    F Foxtrot

    G Golf

    H Hotel

    I India

    J Juliet

    K Kilo

    L Lima

    M Mike

    N November

    O Oscar

    P Papa

    Q Quebec

    R Romeo

    S Sierra

    T Tango

    U Uniform

    V Victor

    W Whiskey

    X X-ray

    Y Yankee

    Z Zulu

  • ALT

    Altitude

  • AM

    Amplitude Modulation

  • AMIL

    Air Mass Impact Line

  • AMMS
